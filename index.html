Projects

College Placement Portal

A web-based placement management system developed as part of a college project. I worked primarily on the backend, focusing on database design, RESTful API development, and notification handling.

The system supports job postings, student applications, job rounds, and real-time notifications using Redis, with MongoDB used for persistent data storage.

Flutter Chat Application

A real-time chat application built using Flutter and Firebase. The application supports one-on-one and group messaging with user authentication, online status, and responsive UI for mobile and web platforms.
